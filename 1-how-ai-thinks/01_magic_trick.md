# 01. The Magic Trick

This one's chat-only. Type each of the three half-finished sentences below into Lumi and see what it picks.

## Prompts

```
To be or not to ____
```

```
Life is like a box of ____
```

```
The world is cruel, but it's also very ____
```

## What's going on

Lumi is guessing the next word from patterns it picked up during training, which is the same thing you do when you complete these in your head. It has just read a lot more sentences than you have. There's no separate "understanding" step happening underneath either side; when Lumi picks something weird, that's the same pattern-matching machinery as when it picks something obvious.
